# EPrints Move Division

 Script for moving all EPrints from one or more divisions to another.

## Installation

Copy move_division to ~eprints/bin/

## Usage

```bash
  $ ./bin/move_division <repository_id> <from...> <to> [--quiet] [--force]
```
